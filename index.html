Welcome to Chat314

Your personal AI assistant using open-source models. Chat314 uses a smart neural network to route your queries to AI models, search engines, or reasoning tools.

How it Works

Chat314 intelligently selects the right AI or search tool using the Classy-AI Python package, making your interactions smoother and faster.

Source Code

Components

Classy

Classy is a Python module designed to classify natural language input into specific categories. It acts as a routing system, sending your questions to the right AI models or tools. Whether it's generating text with Llama, creating images with Dall-E, searching the web with the Google Search API, or just chatting, Classy ensures your query is handled by the most appropriate service.

Google Search API

Chat314 uses the Google Search API to fetch relevant information from the web. To enhance your experience, search queries are generated with the llama-3.1-8b-instant model and formated with the gemma2-9b-it model, making them clear and easy to understand. Whether you’re looking for articles, definitions, or news, Chat314 delivers accurate results efficiently.

API Link
LLM

For creative and conversational tasks, Chat314 uses the meta-llama/llama-4-maverick-17b-128e-instruct and gemma2-9b-it models. You can choose between these models based on your needs — whether you want to write stories, generate lists, brainstorm ideas, or simply chat. The choice is yours.

API Link
Reasoning

For logical reasoning and solving complex math problems, Chat314 uses the deepseek-r1-distill-llama-70b model. It’s designed to tackle challenging questions, making it a reliable assistant for problem-solving, calculations, and step-by-step explanations.

API Link
Dall-E

Dall-E is an AI model capable of generating images from text prompts. Currently, this feature is not available on Chat314. Stay tuned for updates as we work on integrating Dall-E into the platform.
